# Drum-Kit
DESCRIPTION
------------
* It is a simple website for playing various drum sounds.

* It has several buttons with letters and background images of each drum type. When a button is clicked, the corresponding sound is heard with an animation on the button that lasts for 0.1 seconds. This sound and animation is also seen when the key of that particular letter is pressed on the keyboard (both lowercase and uppercase).

* Created using HTML,CSS and Javascript.
